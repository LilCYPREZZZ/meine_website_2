Quiz 05

CSS

Frage 1 von 3
Wofür sind HTML und CSS jeweils zuständig?
HTML ist für Struktur und Inhalt da. CSS kümmert sich um Farben, Schriften und Layout.
Frage 2 von 3
Welche Schreibweise färbt einen einzelnen Absatz rot ein?
Direkt im Tag schreibt man CSS mit dem style-Attribut: style="color:red".
Frage 3 von 3
Warum ist es sinnvoll, CSS in eine eigene Datei auszulagern?
Eine zentrale CSS-Datei spart Arbeit. Du änderst sie einmal, und alle HTML-Seiten, die sie einbinden, sehen sofort anders aus.
