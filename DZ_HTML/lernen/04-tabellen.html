Was ist überhaupt HTML?

Kurz gesagt

Die Sprache, mit der Webseiten gebaut werden

HTML ist im Grunde die Sprache, mit der Webseiten gebaut werden. Alles, was du im Internet siehst, besteht im Hintergrund aus HTML. Es steht für HyperText Markup Language und ist keine typische Programmiersprache, sondern eher eine Art Bauplan, der festlegt, was auf einer Seite erscheint – also zum Beispiel Überschriften, Texte, Bilder oder Links.

Geschichte

Erfunden 1991 von Tim Berners-Lee

HTML wurde 1991 von Tim Berners-Lee erfunden. Seitdem ist es die Basis von fast allen Websites – vom kleinen Blog bis zur großen Plattform. Über die Jahre hat sich HTML weiterentwickelt, aber die Grundidee ist bis heute gleich geblieben.

Was bedeutet das?

Markup statt Programmierung

„Markup" heißt so viel wie auszeichnen. Du schreibst Text und markierst mit kleinen Befehlen (sogenannten <tags>), was er sein soll. Der Browser liest diesen Bauplan und zeigt das Ergebnis an.

  • Überschrift mit <h1>
  • Absatz mit <p>
  • Link mit <a>
  • Bild mit <img>
Merken

Das Wichtigste auf einen Blick

  • HTML = HyperText Markup Language.
  • Es ist keine Programmiersprache, sondern eine Auszeichnungssprache.
  • Es legt fest, was auf einer Seite steht – nicht, wie es aussieht.
  • Erfunden 1991 von Tim Berners-Lee.
  • Grundlage von fast allen Websites im Internet.
