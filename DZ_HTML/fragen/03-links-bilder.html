Quiz 03

Links & Bilder

Frage 1 von 3
Welcher Code bindet ein Bild mit 300 Pixel Breite richtig ein?
Der Tag heißt <img>. Der Dateipfad steht in src und die Breite in width.
Frage 2 von 3
Wozu ist das Attribut alt bei einem Bild gut?
Das Attribut alt ist ein Ersatztext. Er hilft, wenn das Bild fehlt, und wird von Screenreadern vorgelesen.
Frage 3 von 3
Mit welchem Attribut legst du bei einem Link das Ziel fest?
Der Tag <a> nutzt href (hypertext reference), um auf die Zieladresse zu zeigen.
